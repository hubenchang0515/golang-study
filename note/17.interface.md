# 接口
接口(Interface)是一种抽象的自定义类型，用于实现多态。
>多态指同一个实体具有多种表现形式。例如名为A的接口，既可以是B结构体，也可以是C结构体。

接口的定义方式 :  
```Go
type 接口类型名 interface {
    方法名1(参数列表) (返回值列表)
    方法名2(参数列表) (返回值列表)
    方法名3(参数列表) (返回值列表)
    ...
}
```
* 一个变量，只要绑定的方法，涵盖了某个接口的所有方法，就可以赋值给该接口。
* 对于空接口`interface{}`，由于一个方法也没有，因此任何变量都可以赋值给该接口。

例如 :  
```Go
package main

import (
    "fmt"
)

// "可燃物"接口
type Combustible interface {
    ignite() // "点火"方法
}

// "TNT"结构体
type Tnt struct {
}

// "TNT"绑定"点火"方法
func (self Tnt) ignite(){
    fmt.Println("BOOM! You died.")
}

// "烟花"结构体
type Fireworks struct {
}

// "烟花"绑定"点火"方法
func (self Fireworks) ignite(){
    fmt.Println("\\|/")
    fmt.Println("-.-")
    fmt.Println("/|\\")
}

/* 函数名 : ignite
 * 功  能 : 对传入的参数调用点燃方法
 * 参  数 : obj - 待点燃的对象
 * 返回值 : 无
 */
func ignite(obj Combustible){
    obj.ignite()
}

func main(){
    var tnt Tnt
    var fire Fireworks
    
    ignite(tnt)
    ignite(fire)
}
```

指针传递接收器的写法 :  
```Go
package main

import (
    "fmt"
)

// "可燃物"接口
type Combustible interface {
    ignite() // "点火"方法
}

// "TNT"结构体
type Tnt struct {
}

// *Tnt 绑定"点火"方法
func (self *Tnt) ignite(){
    fmt.Println("BOOM! You died.")
}

// "烟花"结构体
type Fireworks struct {
}

// *Fireworks 绑定"点火"方法
func (self *Fireworks) ignite(){
    fmt.Println("\\|/")
    fmt.Println("-.-")
    fmt.Println("/|\\")
}

/* 函数名 : ignite
 * 功  能 : 对传入的参数调用点燃方法
 * 参  数 : obj - 待点燃的对象
 * 返回值 : 无
 */
func ignite(obj Combustible){
    obj.ignite()
}

func main(){
    var tnt Tnt
    var fire Fireworks
    
    ignite(&tnt)   // 取地址
    ignite(&fire)  // 取地址
}
```

空接口`interface{}`可以接受任何类型的变量 :  
```Go
package main

import (
    "fmt"
)

/* 函数名 : test
 * 功  能 : 在切片中插入一个值
 * 参  数 : param - 待插入的切片
 *          pos   - 插入的位置
 *          value - 插入的值
 * 返回值 : 无
 */
func test(param interface{}) {
    fmt.Println(param)
}

func main(){
    var suika [10]int = [10]int {1,2,3,4,5,6,7,8,9,0}
    var bug map[string]int = make(map[string]int)
    
    test(suika)
    test(bug)
}
```
