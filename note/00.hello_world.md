# Hello World

创建源文件`main.go`,写入以下代码 :  
```Go
/* 设置包名 */
package main

/* 导入模块 */
import (
    "fmt"
)

/* 入口函数 */
func main(){
    fmt.Println("hello world")
}
```
* Go的入口函数规定为`main`，程序总是从入口函数开始执行

编译`main.go`,生成可执行文件`runme` :  
```bash
$ go build -o runme
```
* `-o`选项设置生成的可执行文件的名字

运行 :  
```bash
$ ./runme
hello world
```
